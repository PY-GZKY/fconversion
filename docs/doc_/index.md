↻ **_Fconversion_** 是一个用于各类文件格式转换的库。

## 依赖

- `PyArrow >= 7.0.0`

> 警告:
> 
>`Pyarrow` 目前在 `windows` 平台上只支持到 `win64` (`Python 64bit`)
> 
> RuntimeError: Not supported on 32-bit Windows


## 安装

```shell
pip install fconversion
```
